# Sistema de Auditoría y Optimización SEO (Search Engine Optimization) — Versión Élite

Este documento es una guía maestra diseñada para realizar auditorías técnicas y estratégicas orientadas a maximizar la visibilidad en buscadores mediante tácticas de vanguardia (SEO de 2025).

---

## 🚫 Protocolo de Ejecución (OBLIGATORIO)
Se debe seguir este flujo sin excepciones:

1.  **AUDITORÍA Y REPORTE:** Realizar el análisis exhaustivo y entregar el reporte detallado.
2.  **RECOMENDACIONES:** Sugerir mejoras técnicas, de contenido y de autoridad (keywords, intent, E-E-A-T).
3.  **PROHIBICIÓN DE MODIFICACIÓN:** **NO se deben realizar cambios en el código** hasta aprobación explícita.
4.  **APROBACIÓN FASES:** Correcciones solo tras confirmación del usuario.

---

## 📊 Niveles de Madurez SEO
Para asegurar un crecimiento sostenido, las tareas se categorizan según el impacto y la complejidad técnica:

### 🟢 Nivel 1: SEO Básico (Cimientos y Rastreo)
*Objetivo: Que Google encuentre la web y entienda de qué trata.*
- **Indexación:** Robots.txt configurado y Sitemap XML enviado a Search Console.
- **Arquitectura On-Page:** Un solo H1 por página, jerarquía lógica de encabezados (H2-H4).
- **Meta-etiquetas:** Títulos y descripciones únicos con keywords principales y localización.
- **Accesibilidad:** Atributos `alt` en imágenes y enlaces descriptivos.
- **Seguridad:** Certificado SSL (HTTPS) activo.

### 🟡 Nivel 2: SEO Medio (Relevancia y Rendimiento)
*Objetivo: Destacar frente a la competencia y mejorar la experiencia de usuario.*
- **SEO Local Avanzado:** Datos estructurados `LocalBusiness` (JSON-LD) y consistencia NAP.
- **Core Web Vitals:** Optimización de LCP, INP y CLS para una carga fluida.
- **Estrategia de Contenidos:** Creación de un Blog con artículos que ataquen keywords informacionales (Clusters).
- **Enlazado Interno:** Estructura de enlaces para distribuir la autoridad (Link Juice).
- **Optimización de Imágenes:** Uso de formatos WebP/AVIF y lazy-loading.

### 🔴 Nivel 3: SEO Avanzado (Autoridad y Conversión)
*Objetivo: Dominar el mercado, generar confianza (E-E-A-T) y maximizar leads.*
- **E-E-A-T:** Páginas de autor, señales de confianza (reseñas, certificaciones) y políticas claras.
- **SEO Semántico:** Uso de keywords LSI y respuesta directa a la Intención de Búsqueda (Search Intent).
- **CRO + SEO:** Optimización de copies para mejorar el CTR y la tasa de conversión (CTAs persuasivos).
- **SEO Multilingüe:** Sincronización técnica de etiquetas `hreflang` y metadatos en todos los idiomas.
- **Análisis de Canibalización:** Auditoría periódica para evitar que varias páginas compitan por el mismo término.

## 🛠️ Fase 1: Metodología de Auditoría SEO (Por Niveles)

### 1. SEO On-Page (Contenido e Intención)
- **Intención de Búsqueda (Search Intent) [NUEVO]:** Clasificar keywords como: **Informacional, Transaccional, Navegacional o Comercial**. Asegurar que el contenido responde exactamente a esa intención; evitar mezclar múltiples intenciones en una sola URL.
- **Investigación de Keywords:** Identificar términos principales y secundarios (LSI).
- **Meta-Etiquetas de Élite:**
    - **Título y Meta-Descripción:** Optimizar no solo para keywords, sino para el **CTR**. Uso de números, corchetes y verbos de acción.
    - **Fragmentos Enriquecidos (Rich Snippets):** Estrategia de marcación para destacar en la SERP.
- **Jerarquía y Estructura:** Un solo `<h1>` descriptivo y lógica secuencial de encabezados.
- **Optimización de Imágenes:** `alt` descriptivos, nombres de archivo SEO-friendly y formatos de última generación (WebP/AVIF).
- **Contenido Multimedia:** Videos con subtítulos y transcripciones para facilitar la indexación de audio/video.

### 2. Arquitectura de Contenidos y Estructura [NUEVO]
- **Topical Authority (Autoridad Tópica):** Crear **Clusters de Contenido** (Página Pilar + Artículos relacionados).
- **Enlazado Interno:** Estructurar el entrelazado para transferir autoridad (*link juice*) hacia las páginas pilar.
- **Profundidad Tópica:** Cubrir un tema en su totalidad para demostrar expertise ante el algoritmo.

### 3. SEO Técnico y Rastreo
- **Etiquetas Canonical:** Evitar la canibalización y el contenido duplicado.
- **Robots.txt y Meta Robots:** Controlar qué áreas del sitio deben o no ser indexadas.
- **Sitemap XML:** Validar integridad y frecuencia de actualización.
- **Certificado SSL / HTTPS:** Garantizar entorno seguro.
- **Mobile First Indexing:** Comprobar adaptabilidad total y tiempos de carga en móviles.
- **SEO Local:** Consistencia **NAP** y optimización de señales locales.

### 4. Performance y UX (Core Web Vitals)
- **Métricas Críticas:** **LCP** (≤2.5s), **INP** (<200ms) y **CLS** (<0.1).
- **Optimización de Carga:** Compresión, minificación, cache de navegador y CDN.
- **Lazy Loading:** Implementar carga diferida para no penalizar el FCP.

### 5. Autoridad y E-E-A-T (Experience, Expertise, Authoritativeness, Trust) [NUEVO]
- **Señales de Confianza:** Presencia de páginas de "Sobre nosotros", "Contacto" y datos legales claros.
- **Autoría:** Mostrar autores expertos y vinculación con sus perfiles profesionales.
- **Factores de confianza:** Incluir reseñas reales, certificaciones y políticas de privacidad accesibles.
- **Backlinks de Calidad:** Perfil de enlaces entrantes desde sitios con alta autoridad y relevancia temática.

## 🧪 Fase 1.5: Protocolo de Validación Real [NUEVO]
Antes de cerrar el reporte, se debe:
1.  **Validar en Search Console:** Comprobar informes de cobertura, errores de rastreo y validación de páginas indexadas vs enviadas.
2.  **Simular CTR:** Analizar la previsualización del snippet en herramientas de simulación de SERP.
3.  **Auditar Canibalización:** Verificar que dos URLs no compitan por la misma keyword exacta con la misma intención.

## ✅ Checklist de Auditoría SEO (Por Niveles)

### 🟢 Nivel Básico
- [ ] ¿Un solo `<h1>` por página con la keyword principal?
- [ ] ¿Meta-títulos y descripciones únicos y dentro de los límites de caracteres?
- [ ] ¿Atributos `alt` en todas las imágenes siguiendo el patrón descriptivo?
- [ ] ¿Certificado SSL activo y redirección a HTTPS?

### 🟡 Nivel Medio
- [ ] ¿Blog activo con al menos 1 artículo optimizado por trimestre?
- [ ] ¿Datos estructurados (JSON-LD) para negocio local implementados?
- [ ] ¿Velocidad de carga (LCP) inferior a 2.5 segundos?
- [ ] ¿Estrategia de enlazado interno entre artículos del blog y landings de servicio?

### 🔴 Nivel Avanzado
- [ ] ¿Contenido responde al Search Intent (Informacional vs Transaccional)?
- [ ] ¿Señales de E-E-A-T presentes (perfil de autor, contacto, reseñas)?
- [ ] ¿Etiquetas `hreflang` configuradas correctamente para el sitio multilingüe?
- [ ] ¿Copies optimizados para conversión (CTAs persuasivos y directos)?
